# Local File Sharing Application

A Python-based file sharing tool that enables secure and high-speed file transfers across devices on the same local network.

## Features
* **Local Network Sharing:** Transfer files between devices (PC, Mobile, Tablet) on the same WiFi.
* **Concurrency:** Supports multiple file uploads and downloads simultaneously using threading.
* **Web Interface:** Specific user-friendly HTML interface for easy navigation.
* **Automatic IP Detection:** Automatically resolves the host IP address for easy connectivity.

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run the server: `python app.py`
3. Open the browser at the IP address shown in the terminal.
